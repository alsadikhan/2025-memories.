
This project is a bespoke, romantic web application titled "For My Baby - Anniversary Surprise." It is designed as a personalized digital gift, combining interactive elements, media, and heartfelt messages into a single, cohesive experience.
Below is the full technical and functional breakdown of the project.
1. Project Overview
Purpose: An anniversary surprise for a partner ("Princess").
Core Theme: Romantic, soft aesthetics (pink/white palette), and high interactivity.
Access Control: Protected by a "Magic Word" login to maintain privacy and intimacy.
2. Technical Stack
Framework: React 19 (via ES modules/ESM.sh).
Styling: Tailwind CSS for a modern, responsive UI.
Typography:
Inter: For clean, readable body text.
Dancing Script: For a handwritten, romantic feel in headers and notes.
Language: TypeScript (for type-safe components and state management).

4. File Structure Breakdown
index.html: The entry point, loading fonts, Tailwind, and the React import map.
index.tsx: Mounts the React application.
App.tsx: The main controller; manages the high-level state (LOGIN vs. DASHBOARD).
types.ts: Defines the "blueprint" of the app (Interfaces for Songs, Photos, and Notes).
components/:
FloatingHearts.tsx: The background animation engine.
Login.tsx: The password validation logic.
Dashboard.tsx: The layout wrapper and tab-switching logic.
tabs/: Contains the specific content for each section (LoveZone, Music, Gallery, Notes).
5. Design & User Experience (UX)
Color Palette: Uses bg-pink-50 for the background, pink-500 for primary actions, and white glass-morphism effects for cards.
Responsiveness: Fully optimized for mobile devices (essential for a gift often viewed on a phone).
Micro-interactions: Buttons scale down when pressed, cards lift on hover, and transitions between tabs are faded for a "soft" feel.
6. User Flow
Landing: User sees floating hearts and a prompt for a magic word.
Unlock: User enters "princess".
Exploration: User navigates through the tabs, listens to music, views photos, and reads the hidden letter.
Persistence: The "Now Playing" music follows the user even if they switch tabs.
